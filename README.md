## Build And Run

### Installation

1. `git clone ...`
2. `npm install ||npm install --legacy-peer-deps (Ubuntu?)`
3. `npm run local`

Handy commands

- `npm run local`
- `npm run build:dev`
- `npm run test`

To add new elements `directive|pipe|service|class|guard|interface|enum|module`

- `ng generate component home/pages/component-name`
